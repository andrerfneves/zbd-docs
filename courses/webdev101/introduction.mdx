## Course

The WebDev101 Course is meant to be the starting point for anyone hoping to learn about development of web apps and websites. It is a curated collection of video guides and tutorials that will introduce you to basic Web concepts.

## Prerequisites

This course is designed for beginners. No prior knowledge of web development is required.
