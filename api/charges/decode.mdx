## Description

Decodes the inner properties of a given [Charge](/docs/glossary#charge) / Payment Request / Lightning Network invoice.

## Usage

This API is used for further understanding the details of a Charge / Payment Request. The idea is to use this API endpoint if you have to verify/validate any of the individual/specific properties of a Charge / Payment Request.

Any Lightning Network invoice can be used as a property here, even those not generated by the ZEBEDEE API.

</ParamField>

### Body

<ParamField required body="invoice" type="string">
  The Charge / Payment Request / Invoice QR code contents
</ParamField>
