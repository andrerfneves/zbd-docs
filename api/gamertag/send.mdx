---
title: "Send Payment to ZBD Gamertag"
api: "POST https://api.zebedee.io/v0/gamertag/send-payment"
---

## Description

This API endpoint is used to send Bitcoin payments directly to a user's ZBD Gamertag.

## Usage

Unlike other payout endpoints in the ZEBEDEE API, sending payments to ZBD Gamertags is a synchronous action -> this is why there is no `callbackUrl` property. The payment either succeeds or fails on the spot.

## Configuration

### Header Parameters

<ParamField required header="Content-Type" default="application/json" type="application/json" />

<ParamField required header="apikey" type="string">
  ZEBEDEE Project API Key
</ParamField>

### Body

<ParamField required body="amount" type="string">
  The amount for the Payment -> in millisatoshis
</ParamField>

<ParamField required body="gamertag" type="string">
  Destination ZBD Gamertag
</ParamField>

<ParamField body="description" type="string">
  Note or comment for this Payment (visible to recipient)
</ParamField>

<RequestExample>

```bash Request
curl -L -X POST 'https://api.zebedee.io/v0/gamertag/send-payment' \
-H 'Content-Type: application/json' \
--data-raw '{
  "amount": "1000",
  "description": "Sending to ZBD Gamertag",
  "gamertag": "foxp2"
}'
```

</RequestExample>
