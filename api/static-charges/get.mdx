title: "Get Static Charge"

---

## Description

Retrieves all information relating to a specific [Static Charge](/docs/static-charge).

## Usage

You should rely on `callbackUrl` property and expect async Static Charge updates sent from ZEBEDEE servers. However, this API can be used to check any information about any specific Static Charge, specifically properties such as `slots`, `successMessage`, and `status`.

## Configuration

### Path Parameters

<ParamField required path="STATIC_CHARGE_ID" type="string">
  ID of the Static Charge
</ParamField>

### Header Parameters

<ParamField required header="apikey" type="string">
  ZEBEDEE Project API Key
</ParamField>
