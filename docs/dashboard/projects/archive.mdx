---
title: 'Archive Project'
---

If you ever want to archive a Project, archive it by following these steps.

danger

Archiving a project disables the API key.

1.  Navigate to a Project
    
2.  Press the `...` button in the top right of the screen
    
3.  Press `Archive Project`
    

<Frame type="glass" caption="ZEBEDEE Developer Dashboard | Archive Project">
  <img src="/img/v2/projects-archive-project.png" alt="ZEBEDEE Developer Dashboard | Archive Project" />
</Frame>
<br />

4.  Confirm you want to archive the project by pressing `Archive`

<Frame type="glass" caption="ZEBEDEE Developer Dashboard | Archive Project">
  <img src="/img/v2/projects-archive-project-2.png" alt="ZEBEDEE Developer Dashboard | Archive Project" />
</Frame>
<br />

5.  Type `ARCHIVE PROJECT` into the modal and press `Archive` for a final time.

<Frame type="glass" caption="ZEBEDEE Developer Dashboard | Archive Project">
  <img src="/img/v2/projects-archive-project-3.png" alt="ZEBEDEE Developer Dashboard | Archive Project" />
</Frame>
<br />

<Tip>All funds held within a Project Wallet when it's archived will be transferred automatically to your Developer Wallet.</Tip>

Great! You've successfully archived your project. You can always [create another one!](/docs/dashboard/projects/create)
