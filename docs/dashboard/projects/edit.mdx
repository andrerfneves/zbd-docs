You can edit a Projectâ€™s Name, Description, and Image. There are two ways to edit a project:

## 1. Projects View

From the `Projects` page, find your Project, click the `...` on the Project card, and click `Edit Project`.

## 2. Single Project View

From within a `Project` page, press `...` menu on the top right corner and choose `Edit Project Info`.
