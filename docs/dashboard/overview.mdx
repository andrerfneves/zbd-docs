## What is the Developer Dashboard?[​](#what-is-the-developer-dashboard "Direct link to heading")

The [ZEBEDEE Developer Dashboard](https://dashboard.zebedee.io) is your portal for integrating with next-gen payments via the ZEBEDEE API.

It allows you to:

*   Manage a fully-programmable Bitcoin Lightning Network wallet
*   Create and manage your Projects
*   Access API key(s) to interact with our powerful payment endpoints
*   Oversee transaction activity
*   Create single-time vouchers that you can share with others
*   And more!
