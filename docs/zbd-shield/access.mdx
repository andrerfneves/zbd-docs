# Access ZBD Shield

Each ZBD Shield instance runs on a unique domain and requires unique login credentials. These are different login credentials than your [ZEBEDEE Developer Dashboard](https://dashboard.zebedee.io) account.

1.  To access your ZBD Shield account, navigate to the URL link provided by the ZEBEDEE team (it should resemble: `https://{yourShieldDomain}.zebedeepowered.com`)
    
2.  Once on the `Login` screen, use the provided `username` and `password` to sign in.
