# ZEBEDEE App URI Schemes

If you wish to open or trigger the ZEBEDEE App on iOS and Android devices, you may use the following URI schemes:

```
zbd
zeb
lnurl
zebedee
lightning
zebedeeapp
zebedeewallet
```

To use the URI schemes above you may follow a URL similar to the ones below:

```
# Opening a Bitcoin Lightning Network invoice
lightning:lnbc109......71fe

# Opening a LNURL QR code
lnurl:lnurl1dp68gurn...e3c3svx0h

# Opening ZEBEDEE App with Bitcoin Lightning Network invoice
zebedee:lightning:lnbc109......71fe

# Opening ZEBEDEE App with redeeming Voucher
zebedee:voucher:XXXXXXXX
```
