## Usage

This API endpoint is the reverse of the [Get User ID for ZBD Username](/api-reference/username/get-user-id) endpoint. Use this endpoint to get a given user's ZBD Username.

<Tip>If you've got a User ID and you wish to find out that user's ZBD Username, perhaps to then [send that user payments](/api-reference/gamertag/send), you may use this endpoint.</Tip>

## Configuration

### Path Parameters

<ParamField required path="id" type="string">
  User ID
</ParamField>

### Header Parameters

<ParamField required header="apikey" type="string">
  ZBD Project API Key
</ParamField>
