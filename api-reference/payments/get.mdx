## Description

Retrieves all the information related to a specific Payment.

## Usage

Use this API endpoint to get all of the properties of a given Payment record. While the proper way is to use the `callbackUrl` property to receive Payment updates, you may use this API endpoint and fetch the Payment `status`, for example.

## Configuration

### Path Parameters

<ParamField required path="PAYMENT_ID" type="string">
  PAYMENT ID
</ParamField>

### Header Parameters

  ZBD Project API Key
</ParamField>
