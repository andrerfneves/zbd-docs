## Description

In order to get data on payments sent to ZBD Gamertags, you can use this endpoint. The data payload returned will inform you of the `status` of that transaction as well as any associated `fees`.

## Usage

The idea here is to use the `transactionId` property returned from the [Pay to ZBD Gamertag](/api-reference/zbd-gamertag/send) response, and use that ID to fetch the latest details of the given Payment.

## Configuration

### Path Parameters

<ParamField required path="id" type="string">
  Payment Transaction ID
</ParamField>

### Header Parameters

<ParamField required header="apikey" type="string">
  ZBD Project API Key
</ParamField>
